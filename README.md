# Vietnam Travel Guide - Web Application
=================================================

This is a web application for exploring tourism in Vietnam. It helps users discover beautiful places and local specialties across different provinces through an interactive map.

### **Tech Stack**
-----------------------------------
* **HTML5** and **CSS3** are used to build the website structure and layout.
* **JavaScript** handles the interactive map clicking actions.
* **PHP** is used for the basic user authentication system.

### **Key Features**
-----------------------------------
* **Interactive Vietnam Map:** Users can click on different provinces on the map to see specific travel information and local food specialties.
* **Province Information:** Displays detailed data about famous tourist attractions and traditional food of each region.
* **User Authentication:** Includes a functional login system built with PHP.
* **Future Plans:** Designed to support ticket booking and travel reservation features in later updates.

### **Installation & Setup Guide**
-----------------------------------
1. Download or clone this project to your local computer.
2. Open the `index.html` file in any web browser to view the frontend interface.
3. To use the login feature, run the project on a local server environment like **XAMPP** and configure the PHP files.
